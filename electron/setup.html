Configuração inicial

Como você quer usar o Memory MCP?

Escolha o modo de operação. Você pode trocar isso depois em Configurações.

🖥️
Local
Roda tudo no seu PC. Precisa de uma URL de banco PostgreSQL (grátis no Neon).
☁️
VPS / Cloud
Conecta a um servidor já existente no EasyPanel, Railway, Render, etc.

URL do dashboard

Cole a URL onde o painel do Memory MCP abre no seu browser. Ao abrir, vai aparecer uma tela de login.

💡 Essa URL precisa mostrar o painel (interface web), não a API.
No EasyPanel, é a URL do serviço que tem o frontend ou que usa o Dockerfile da raiz do projeto.

Se você só tem o backend deployado, veja como deployer o frontend →
A URL deve abrir uma tela de login quando acessada no browser.

Configurar servidor local

O servidor vai rodar direto no seu PC. Preencha as chaves abaixo.

📦 O banco de dados é o único serviço externo necessário.
Crie um grátis em neon.tech → New Project → copie a connection string.
Neon.tech oferece 512MB grátis — sem cartão de crédito. Criar conta grátis →
Usado para embeddings semânticos. Gerar chave →
Pesquisa web inteligente. 1000 buscas/mês grátis →
Cache de embeddings — melhora performance. Upstash grátis (10k req/dia) →
Se deixar em branco, o sistema funciona normalmente sem cache.
Porta onde o servidor vai rodar. Padrão: 3100

Criar conta de administrador

Essa conta vai ter acesso total ao sistema.

Pronto! ✨

O Memory MCP vai conectar ao seu servidor em instantes.
